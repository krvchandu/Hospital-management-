# 🏥 Hospital Management System in C

A simple **Hospital Management System** built using **C Programming**.  
This project allows users to manage patient records using a **menu-driven interface** and **file handling**.

The system helps in storing, retrieving, and searching patient information efficiently.

---

## 📌 Features

- ➕ Add new patient records
- 📋 Display all patient records
- 🔍 Search patient by ID
- 📊 Find patient with the highest age
- 💾 File-based data storage
- 🖥 Menu-driven interface

---

## 🛠 Technologies Used

- C Programming
- File Handling
- Structures
- Header Files
- Modular Programming
